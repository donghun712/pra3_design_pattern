# 디자인 패턴 실습 요약

# 빌더 패턴
User 객체를 단계적으로 생성할 수 있습니다.
ID, 이름, 이메일을 입력 후 "생성" 버튼을 누르면 사용자가 만들어집니다.
생성된 사용자 정보가 바로 화면에 표시됩니다.

# 브릿지 패턴
메모리와 로컬스토리지 저장소를 같은 방식으로 다룹니다.
저장소 종류(💾/💿)를 선택해 사용자 데이터를 추가, 조회, 삭제할 수 있습니다.
저장소를 바꿀 때마다 해당 저장소의 데이터만 보이며, 불러오기와 초기화도 지원합니다.

# 상태 패턴
사용자의 활성/비활성 상태에 따라 동작이 바뀝니다.
사용자 정보를 입력해 객체를 만든 후, "활성화" 또는 "비활성화" 버튼으로 상태를 변경할 수 있습니다.
현재 상태(활성/비활성)는 화면에 즉시 반영됩니다.
